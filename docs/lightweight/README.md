# DoctorPet 경량 문서

이 디렉터리의 문서는 팀원이 빠르게 열람하기 위한 요약본이다. 구현과 정책 판단은 각 문서 상단에 표시된 저장소 정본을 따른다. 경량본과 정본이 다르면 `PRD → SA → 코드 컨벤션 → 정책 정리본` 순으로 적용하고 충돌 사실을 먼저 확인한다.

| 문서 | 용도 |
| --- | --- |
| [프로젝트 정책 경량본](DoctorPet-정책-경량본.md) | 회원·병원·예약·노쇼·결제·AI 정책 요약 |
| [PRD 경량본](DoctorPet-PRD-경량본.md) | 제품 목표·핵심 기능·MVP 범위·성공 기준 요약 |
| [SA 경량본](DoctorPet-SA-경량본.md) | 시스템 구성·핵심 설계·상태 전이·스케줄러 요약 |
| [DB 설계 경량본](DoctorPet-DB설계-경량본.md) | 관계·테이블·제약·인덱스 요약 |

검색 Tool 실패 응답의 생성 주체는 현재 정본 간 충돌로 `[결정 필요]` 상태다. 팀 합의 후 정본을 먼저 수정하고 경량본을 함께 갱신한다.
